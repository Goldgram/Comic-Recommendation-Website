What is a Comic Book Recommender?

A comic book recommendation system takes information from the user for their preference of comic book Series, Writers, Artists and Characters. The information is taken into the system and returns a list of Comic Book Series that the user might be interested in. With these results the user can view further information, give feedback if they like/dislike the results and add more inputs into the system. The recommender then can take in this data and reproduce better result for the user, this can be done an infinite amount of times to give the user the most accurate results.
